indet    --    individual indeterminates


Syntax 
indet(R: RING, N: INT): RINGELEM

Description
This function returns the N-th indeterminate of the current ring.

Example 
/**/  use R ::= QQ[x,y,z];
/**/  indet(R, 2);
y
