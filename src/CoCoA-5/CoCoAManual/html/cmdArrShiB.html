ArrShiB    --    Shi arrangement of type B


Syntax 
ArrShiB(S: RING, k: INT): LIST

Description
This function constructs the Shi arrangement of type B in the first k variables of the polynomial ring S.

Example 
/**/ use S::=QQ[x,y,z];	
/**/ ArrShiB(S, 2);
[x,  y,  x -y,  x +y,  x -1,  y -1,  x -y -1,  x +y -1]
