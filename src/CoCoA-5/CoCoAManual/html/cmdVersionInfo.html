VersionInfo    --    version and info about CoCoA


Syntax 
VersionInfo(): RECORD

Description
This function returns a record with various information about CoCoA and CoCoALib (the mathematical core of CoCoA)

Example 
/**/ indent(VersionInfo(),2);
record[
  CoCoALibVersion := "0.99***",
  CoCoAVersion := "5.*.*",
  CompilationDate := ....,
  ...
  ExternalLibs := [...]
]
