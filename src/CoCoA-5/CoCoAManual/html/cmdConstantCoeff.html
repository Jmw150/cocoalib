ConstantCoeff    --    constant coefficient of a polynomial


Syntax 
ConstantCoeff(F: RINGELEM): RINGELEM

Description
This function returns the constant coefficient of a polynomial. The result is in the coefficient ring, and may be zero.

Example 
/**/  use QQ[x,y];
/**/  f := x^3+3*x*y-4*y+5;
/**/  ConstantCoeff(f);
5
