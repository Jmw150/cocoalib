concat    --    concatenate lists


Syntax 
concat(L_1: LIST,...,L_n: LIST): LIST

Description
This function returns the list obtained by concatenating the lists L_1,...,L_n.

NOTE: to concatenate strings use ConcatStrings or str1+str2.

Example 
/**/  concat([1,2,3],[4,5],[],[6]);
[1, 2, 3, 4, 5, 6]
