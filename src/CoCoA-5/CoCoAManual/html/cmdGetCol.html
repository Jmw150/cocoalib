GetCol    --    convert a column of a matrix into a list


Syntax 
GetCol(M: MAT, K: INT): LIST

Description
This function makes a list containing the entries of the K-th column of M.

Example 
/**/  M := mat([[1,2], [3,4]]);
/**/  GetCol(M,2);
[2, 4]
