IsPosetGraded    --    checks if the poset is graded


Syntax 
IsPosetGraded(relP: LIST): BOOL

Description
This function tests whether the poset P is graded from the list relP of its strict relations.

Example 
// POSET:
//         3   4
//          \ /
//           2
//           |
//           1
/**/ relP := [[1,  2], [2, 3],  [2,  4]];    
/**/ IsPosetGraded(relP);
true
    
// P:
//           5
//         /  \   
//        3    4
//        |    |
//         2   |
//          \ /
//           1
/**/ relP := [[1,2],  [2,3],  [1,4],  [3,5],  [4,5]];
/**/ IsPosetGraded(relP);
false
