ContFrac    --    continued fraction quotients


Syntax 
ContFrac(X: RAT): LIST of INT

Description
ContFrac returns a list of the continued fraction quotients for the given rational number X.

Example 
/**/  ContFrac(1.414213);
[1, 2, 2, 2, 2, 2, 2, 2, 1, 1, 4, 1, 1, 1, 1, 1, 2, 1, 6]
