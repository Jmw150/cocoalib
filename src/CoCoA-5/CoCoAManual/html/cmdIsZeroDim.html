IsZeroDim    --    test whether an ideal is zero-dimensional


Syntax 
IsZeroDim(I: IDEAL): BOOL

Description
This function tests whether its argument is zero-dimensional.

Example 
/**/  use QQ[x,y,z];
/**/  IsZeroDim(ideal(x));
false
/**/  IsZeroDim(ideal(x^3, y^4-x ,z-3));
true
/**/  IsZeroDim(ideal(x^2, x*y^3));
false
