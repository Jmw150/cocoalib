ConcatLists    --    concatenate a list of lists


Syntax 
ConcatLists(L: LIST of LISTs): LIST

Description
This function takes one argument, a list whose entries are lists, and returns the concatenation of its entries.

Example 
/**/  L := [[1,2],["abc","def"],[3,4]];
/**/  ConcatLists(L);
[1, 2, "abc", "def", 3, 4]
