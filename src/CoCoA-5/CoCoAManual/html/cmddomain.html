domain    --    domain of a homomorphism


Syntax 
domain(phi: RINGHOM): RING

Description
This function returns the domain of the homomorphism phi.

Example 
/**/  P := NewPolyRing(RingQQ(), "alpha,beta");
/**/  phi := CanonicalHom(RingZZ(), P);
/**/  domain(phi);
ZZ
/**/  psi := CoeffEmbeddingHom(P);
/**/  domain(psi);
QQ
