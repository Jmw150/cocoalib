abs    --    absolute value of a number


Syntax 
abs(N: INT): INT
abs(N: RAT): RAT
abs(N: RINGELEM): RINGELEM

Description
This function returns the absolute value of N. If N is a RINGELEM then it must belong to an ordered ring.

Example 
/**/  abs(-3);
3

/**/  abs(-2/3);
2/3
