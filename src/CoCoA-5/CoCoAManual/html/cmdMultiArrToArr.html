MultiArrToArr    --    underling arrangement from a multiarrangement


Syntax 
MultiArrToArr(MultiA: LIST): LIST

Description
This function constructs the underling arrangement of the multiarrangement MultiA.

Example 
/**/ use QQ[x,y,z];
/**/ MultiA := [[x,1], [y,3], [z,2]];
/**/ MultiArrToArr(MultiA);
[x,  y,  z]
