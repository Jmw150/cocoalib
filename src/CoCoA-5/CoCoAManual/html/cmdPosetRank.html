PosetRank    --    rank of the poset from its relations


Syntax 
PosetRank(relP: LIST): INT

Description
This function returns the rank of a poset from the list relP of its strict relations.

Example 
// POSET:
//         3   4
//          \ /
//           2
//           |
//           1
/**/ relP := [[1,  2], [2, 3],  [2,  4]];
/**/ PosetRank(relP);
2
