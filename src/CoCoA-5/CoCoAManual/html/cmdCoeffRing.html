CoeffRing    --    the ring of coefficients of a polynomial ring


Syntax 
CoeffRing(R: RING): RING

Description
This function returns the ring of coefficients of a polynomial ring.

Example 
/**/  use R ::= QQ[x,y,z];
/**/  S ::= ZZ/(2)[a,b,c];
/**/  CoeffRing(R);
QQ

/**/  CoeffRing(S);
FFp(2)
