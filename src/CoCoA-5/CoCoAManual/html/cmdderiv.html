deriv    --    the derivative of a polynomial or rational function


Syntax 
deriv(F: RINGELEM, X: RINGELEM): RINGELEM

Description
This function returns the derivative of F with respect to the indeterminate X.

Example 
/**/  use R ::= QQ[x,y];
/**/  deriv(x*y^2, x);
y^2

/**/  FrF := NewFractionField(R);
/**/  use FrF;
/**/  deriv((x*y^2)/(x-1), x);
(-y^2)/(x^2 -2*x +1)
