IsCoprime    --    checks if t1 is coprime with t2


Syntax 
IsCoprime(t1: RINGELEM, t2: RINGELEM): BOOL

Description
This function tests whether two POWER-PRODUCTS t1 and t2 are coprime. (".. is coprime with ..").

Example 
/**/ use QQ[x,y,z];
/**/ IsCoprime(x*y, y*z);
false
/**/ IsCoprime(x*y, z);
true
