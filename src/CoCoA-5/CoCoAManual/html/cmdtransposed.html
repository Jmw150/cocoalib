transposed    --    the transposition of a matrix


Syntax 
transposed(M: MAT): MAT

Description
This function returns the transpose of the matrix M.

Example 
/**/  M := mat([[1,2,3],[4,5,6]]);
/**/  M;
matrix(QQ,
 [[1, 2, 3],
  [4, 5, 6]])

/**/  transposed(M);
matrix(QQ,
 [[1, 4],
  [2, 5],
  [3, 6]])
