IsAtEOF    --    checks if input stream has reached end of input


Syntax 
IsAtEOF(in: ISTREAM): BOOL

Description
This function tests whether the input stream in has reached end of input.

Example 
/**/  Istring := OpenIString("just one line");
/**/  GetLine(Istring);
just one line
/**/  IsAtEOF(Istring);
true
