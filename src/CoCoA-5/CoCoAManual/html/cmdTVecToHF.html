TVecToHF    --    Hilbert Function of a type vector


Syntax 
TVecToHF(TV: LIST): TAGGED

Description
This function returns the Hilbert function associated to the type vector TV.

See TVecFromHF

Example 
/**/  Use P ::= QQ[x,y,z,t];
/**/  T := TVecFromHF([[1, 3, 5], 6]);  T;
[[[2],  [4]]]
/**/  TVecToHF([[[2], [4]]]);
H(0) = 1
H(1) = 3
H(2) = 5
H(t) = 6   for t >= 3
