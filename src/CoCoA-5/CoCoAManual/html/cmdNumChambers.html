NumChambers    --    number of chambers of an arrangement of hyperplanes


Syntax 
NumChambers(A: LIST): INT

Description
This function returns the number of chambers from the list A of hyperplanes in the arrangement.

Example 
/**/ use QQ[x,y];	
/**/ A := [x, x-y, y];
/**/ NumChambers(A);
6
