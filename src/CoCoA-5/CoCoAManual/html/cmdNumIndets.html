NumIndets    --    number of indeterminates


Syntax 
NumIndets(R: RING): INT

Description
This function returns the number of indeterminates of the ring R.

Example 
/**/  S ::= QQ[x,y];
/**/  R ::= QQ[x,y,z];
/**/  NumIndets(R);
3
/**/  NumIndets(S);
2
