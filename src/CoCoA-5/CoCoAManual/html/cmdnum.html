num    --    numerator


Syntax 
num(N: INT): INT
num(N: RAT): INT
num(N: RINGELEM): RINGELEM

Description
This function returns the numerator of N.

The OBSOLETE fragile syntax in CoCoA 4 N.Num and N.Den is no longer supported.

Example 
/**/  num(3);
3

/**/  P ::= QQ[x,y];
/**/  F := NewFractionField(P);
/**/  use F;
/**/  num(x/(x+y));
x
