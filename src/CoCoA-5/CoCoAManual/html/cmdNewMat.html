NewMat    --    Zero matrix


Syntax 
NewMat(R: RING, M: INT, N: INT): MAT

Description
This function is kept for CoCoA-4 nostalgia: better use ZeroMat .

Example 
/**/  use S ::= QQ[x,y,z];
/**/  NewMat(S,2,3);
matrix( /*RingDistrMPolyClean(QQ, 3)*/
 [[0, 0, 0],
  [0, 0, 0]])
/**/  ZeroMat(S,2,3);
matrix( /*RingDistrMPolyClean(QQ, 3)*/
 [[0, 0, 0],
  [0, 0, 0]])
