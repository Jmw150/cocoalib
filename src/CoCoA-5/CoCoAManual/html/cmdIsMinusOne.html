IsMinusOne    --    test whether an object is -1


Syntax 
IsOne(X: OBJECT): BOOL

Description
This function tests whether its argument is -1; the argument can be of almost any type for which -1 makes sense.

Example 
/**/  IsMinusOne(23);
false
/**/  IsMinusOne(3/-3);
true
/**/  use R ::= QQ[x,y,z];
/**/  IsMinusOne(-x/x);
true
