power    --    compute a power


Syntax 
power(X: INT, E: INT): INT
power(X: RAT, E: INT): RAT
power(X: RINGELEM, E: INT): RINGELEM
power(X: MAT, E: INT): MAT

Description
This function calculates powers; it is the same as the infix "hat" operator.

Example 
/**/  power(2,3);
8
/**/  use QQ[x];
/**/  power(x+1,2);
x^2 +2*x +1
