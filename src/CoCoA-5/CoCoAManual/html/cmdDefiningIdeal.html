DefiningIdeal    --    defining ideal of a quotient ring


Syntax 
DefiningIdeal(S: RING): IDEAL

Description
When S is a quotient ring, say S = R/I, this function returns I, the ideal which defines S.

Example 
/**/  use R ::= QQ[x,y,z];
/**/  S := R/ideal(x);
/**/  DefiningIdeal(S);
ideal(x)
