diff    --    returns the difference between two lists


Syntax 
diff(L: LIST, M: LIST): LIST

Description
This function returns the list obtained by removing all the elements of M from L. See deriv for the derivative.

Example 
/**/  L := [1,2,3,2,[2,3]];
/**/  M := [1,2];
/**/  diff(L, M);
[3, [2, 3]]
