XelMat    --    matrices for std. term-orderings


Syntax 
XelMat(N: INT): MAT

Description
This function returns the matrix defining a standard term-ordering.

Example 
/**/  XelMat(3);
matrix(ZZ,
 [[0, 0, 1],
  [0, 1, 0],
  [1, 0, 0]])
