IsZZ    --    test whether a ring is the ring of integers


Syntax 
IsZZ(R: RING): BOOL

Description
This function tests whether a ring is the ring of integers.

Example 
/**/  R ::= QQ[x,y];
/**/  IsZZ(CoeffRing(R));
false
/**/  IsZZ(BaseRing(CoeffRing(R)));
true
