StdDegRevLexMat    --    matrix for std deg rev lex term-ordering


Syntax 
StdDegRevLexMat(N: INT): MAT

Description
This function returns the matrix defining a standard term-ordering.

Example 
/**/  StdDegRevLexMat(3);
matrix(ZZ,
 [[1, 1, 1],
  [0, 0, -1],
  [0, -1, 0]])
