LexMat    --    matrices for std. term-orderings


Syntax 
LexMat(N: INT): MAT

Description
This function returns the matrix defining the standard term-ordering "lex".

Example 
/**/  LexMat(3);
matrix(ZZ,
 [[1, 0, 0],
  [0, 1, 0],
  [0, 0, 1]
])
