MakeTerm    --    returns a monomial (power-product) with given exponents


Syntax 
MakeTerm(R: RING, L: LIST of INT): RINGELEM

Description
This function returns the power-product in R whose list of exponents is L. It is the inverse of exponents . The length of L must be equal to the number of indeterminates in R.

This function was called LogToTerm up to version CoCoA-5.1.2.

Example 
/**/  use R ::= QQ[x,y,z];
/**/  MakeTerm(R, [2,3,5]);
x^2*y^3*z^5

/**/  exponents(It);
[2, 3, 5]
