GetCols    --    convert a matrix into a list of lists


Syntax 
GetCols(M: MAT): LIST of LIST

Description
This function produces a list of lists containing the columns of M.

Example 
/**/  M := mat([[1,2], [3,4]]);
/**/  GetCols(M);
[[1, 3], [2, 4]]
