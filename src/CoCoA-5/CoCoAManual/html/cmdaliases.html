aliases    --    list of global aliases


Syntax 
aliases(): TAGGED("aliases")

Description
This function prints a list of global aliases for packages. Aliases are formed with the function alias .

Example 
/**/  alias LL := $abcd;
/**/  aliases();

Coclib       = $coclib
Approx       = $approx
(...)
TP           = $contrib/thmproving
TV           = $contrib/typevectors
LL           = $abcd
