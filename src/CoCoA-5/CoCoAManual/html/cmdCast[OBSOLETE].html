Cast [OBSOLETE]    --    [OBSOLETE] type conversion

Description
This old function is now obsolete. These are the functions for casting:

To cast INT, RAT, STRING to a polynomial (and more in general to a RINGELEM) use RingElem .

To cast RINGELEM to INT, RAT use AsINT , AsRAT .

To cast LIST to MAT use matrix . To cast MAT to LIST use GetRows , GetCols .

To cast a MODULEELEM to LIST use compts .

To cast a MODULE to MAT use GensAsCols, GensAsRows . To cast a MAT to MODULE use SubmoduleCols, SubmoduleRows .
