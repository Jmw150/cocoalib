ceil    --    round rational up to integer


Syntax 
ceil(X: RAT): INT

Description
This function returns the least integer greater than or equal to X.

Example 
/**/  ceil(0.99);
1

/**/  ceil(0.01);
1

/**/  ceil(1);
1

/**/  ceil(-0.99);
0
