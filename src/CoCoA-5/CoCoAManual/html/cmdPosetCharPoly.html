PosetCharPoly    --    characteristic polynomial of a poset from the relations of the poset


Syntax 
PosetCharPoly(relP: LIST): RINGELEM

Description
This function returns the characteristic polynomial from the list relP of the strict relations in a graded poset in the variable t.

Example 
// POSET:
//         3   4
//          \ /
//           2
//           |
//           1
/**/ relP := [[1,  2], [2, 3],  [2,  4]];
/**/ PosetCharPoly(relP);
t^2 -t
