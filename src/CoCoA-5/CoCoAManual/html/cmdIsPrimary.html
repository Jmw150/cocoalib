IsPrimary    --    primary test


Syntax 
IsPrimary(I: IDEAL): BOOL

Description
This function determines whether an ideal is primary.

Example 
/**/  use P ::= QQ[x,y,z];
/**/  IsPrimary(ideal(x^2, y^2, z));
true 
/**/  IsPrimary(ideal(x*(x-1), y^2, z));
false
