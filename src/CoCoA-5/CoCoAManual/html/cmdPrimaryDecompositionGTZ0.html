PrimaryDecompositionGTZ0    --    primary decomposition of a 0-dimensional ideal


Syntax 
PrimaryDecompositionGTZ0(I: IDEAL): LIST of IDEAL

Description
This function, alternative to PrimaryDecomposition , returns the primary decomposition of the 0-dimensional ideal I using the GTZ algorithm (implemented by Luis David Garcia -- updated to CoCoA-5 by Anna M. Bigatti).

Example 
/**/  use R ::= QQ[x,y,z];
/**/  PD := PrimaryDecompositionGTZ0(ideal(x-z, y^2-z^2, z^2));
/**/  indent(PD);
