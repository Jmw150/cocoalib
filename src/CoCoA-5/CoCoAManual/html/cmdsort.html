sort    --    sort a list


Syntax 
sort(ref L: LIST)

Description
This procedure sorts the elements of the list in L with respect to the default comparisons related to their types; it overwrites L and returns nothing.

For more on the default comparisons, see Relational Operators in the chapter on operators. For sorting with respect to your own ordering, see SortBy , SortedBy .

Example 
/**/  L := [3,2,1];
/**/  sort(ref L);  -- this returns nothing and modifies L
/**/  L;
[1, 2, 3]

/**/  use R ::= QQ[x,y,z];
/**/  L := [x,y,z];
/**/  sort(ref L);  -- this returns nothing and modifies L
/**/  L[1];
z

/**/  sorted([y, x, x^2]);   -- this returns the sorted list
[y, x, x^2]
