OrdMat    --    matrix defining a term-ordering


Syntax 
OrdMat(R: RING): MAT

Description
This function returns a matrix which describes the term-ordering of the ring R.

Example 
/**/  use S ::= QQ[x,y,z];
/**/  M := mat([ [1,2,3], [3,4,5], [0,0,1]]);
/**/  P := NewPolyRing(CoeffRing(S), IndetSymbols(S), M, 2);
/**/  GradingDim(P);
2
/**/  OrdMat(P);
matrix(QQ,
 [[1, 2, 3],
  [3, 4, 5],
  [0, 0, 1]])

/**/  GradingDim(S);
1
/**/  OrdMat(S);
matrix(QQ,
 [[1, 1, 1],
  [0, 0, -1],
  [0, -1, 0]])
