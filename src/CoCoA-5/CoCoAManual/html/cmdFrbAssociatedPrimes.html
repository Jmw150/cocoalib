FrbAssociatedPrimes    --    Associated primes of monomial ideals


Syntax 
FrbAssociatedPrimes(I: IDEAL): LIST

Description
Using the Frobby C++ library by Bjarke Roune.

Example 
/**/  use R ::= QQ[x,y,z];
/**/  I := ideal(x^2, x*y, y^2, z^2);
/**/  FrbAssociatedPrimes(I);
[ideal(x, y, z)]
