AffHilbertFn    --    the affine Hilbert function


Syntax 
AffHilbertFn(R: (Poly or Quotient)RING): TAGGED("$hp.Hilbert")
AffHilbertFn(R: (Poly or Quotient)RING, N: INT): INT

Description
The first form of this function computes the affine Hilbert function for R. The second form computes the N-th value of the affine Hilbert function. The weights of the indeterminates of R must all be 1. For evaluating of the Hilbert function repeatedly, use the function EvalHilbertFn instead of AffHilbertFn(R, N) in order to speed up execution.

The coefficient ring must be a field.

Example 
/**/  use R ::= QQ[x,y,z];
/**/  AffHilbertFn(R/ideal(z^4-1, x*z^4-y-3));
H(0) = 1
H(1) = 3
H(t) = 4t - 2   for t >= 2
