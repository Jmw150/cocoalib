IsQuotientRing    --    test whether a ring is a quotient ring


Syntax 
IsQuotientRing(R: RING): BOOL

Description
This function tests whether a ring is a quotient ring; it returns true if the ring is a quotient ring.

Example 
/**/  use R ::= QQ[x,y];
/**/  S := R/ideal(x);
/**/  IsQuotientRing(S);
true;
