IsMaximal    --    maximality test


Syntax 
IsMaximal(I: IDEAL): BOOL

Description
This function determines whether an ideal is maximal.

Example 
/**/  use P ::= QQ[x,y,z];
/**/  IsMaximal(ideal(x^2+1, z^3+z-1, x-5*y+4*z));
true 
/**/  IsMaximal(ideal(x^2+1, z^3+z-1, (x-5*y+4*z)*(y-3)));
false
