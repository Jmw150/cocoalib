IsStable    --    checks if an ideal is stable


Syntax 
IsStable(I: IDEAL): BOOL

Description
This function tests whether the monomial ideal I is stable.

Example 
/**/  use R ::= QQ[x,y,z];
/**/  I := ideal(x*y^3, y^4, x^3, x^2*y, x^2*z);
/**/  IsStable(I);
true
