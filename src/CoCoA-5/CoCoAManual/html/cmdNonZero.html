NonZero    --    remove zeroes from a list


Syntax 
NonZero(L: LIST|MODULEELEM): LIST

Description
This function returns the list obtained by removing the zeroes from L.

Example 
/**/  use R ::= QQ[x,y,z];
/**/  NonZero([0,0,3, ideal(y),0]);
[3, ideal(y)]
