ArrShiA    --    Shi arrangement of type A


Syntax 
ArrShiA(S: RING, k: INT): LIST

Description
This function constructs the Shi arrangement of type A in the first k variables of the polynomial ring S.

Example 
/**/ use S::=QQ[x,y,z];	
/**/ ArrShiA(S, 3);
[x -y,  x -z,  y -z,  x -y -1,  x -z -1,  y -z -1]
