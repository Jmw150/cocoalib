ArrShiD    --    Shi arrangement of type D


Syntax 
ArrShiD(S: RING, k: INT): LIST

Description
This function constructs the Shi arrangement of type D in the first k variables of the polynomial ring S.

Example 
/**/ use S::=QQ[x,y,z];	
/**/ ArrShiD(S, 2);
[x -y,  x +y,  x -y -1,  x +y -1]
