IsAntiSymmetric    --    checks if a matrix is anti-symmetric


Syntax 
IsAntiSymmetric(M: MAT): BOOL

Description
This function tests whether the square matrix M is anti-symmetric.

Example 
/**/  M := mat([[0, 1, 2], [-1, 0, 3], [-2, -3, 0]]);
/**/  IsAntiSymmetric(M);
true
