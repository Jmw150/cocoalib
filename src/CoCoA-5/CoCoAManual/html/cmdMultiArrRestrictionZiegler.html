MultiArrRestrictionZiegler    --    Ziegler multirestriction of the arrangement of hyperplanes wrt a hyperplane


Syntax 
MultiArrRestrictionZiegler(A: LIST, H: RINGELEM): LIST

Description
This function returns the Ziegler multirestriction of the arrangement of hyperplanes A with respect to its hyperplane H in the variables [y[1],..,y[n]].

Example 
/**/ use QQ[x,y,z];	
/**/ A := [x, x-z, y-z, z];
/**/ MultiArrRestrictionZiegler(A, z);
[[y[1],  2],  [y[2],  1]]
