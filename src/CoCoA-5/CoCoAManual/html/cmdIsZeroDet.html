IsZeroDet    --    test whether determinant is zero


Syntax 
IsZeroDet(M: MAT): BOOL

Description
This function tests whether the determinant of M is zero; it may sometimes be faster than computing the determinant.

Example 
/**/  M := matrix([[1,2], [3,4]]);
/**/  IsZeroDet(M);
false
