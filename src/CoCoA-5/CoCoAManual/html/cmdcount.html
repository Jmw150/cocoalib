count    --    count the objects in a list


Syntax 
count(L: LIST, E: OBJECT): INT

Description
This function counts the number of occurrences of the object E in the list L.

Example 
/**/  L := [1,2,3,2,[2,3]];
/**/  count(L,2);
2

/**/  count(L,[2,3]);
1

/**/  count(L,"a");
0
