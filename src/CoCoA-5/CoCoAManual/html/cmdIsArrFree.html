IsArrFree    --    checks if the arrangement is free


Syntax 
IsArrFree(Q: RINGELEM): BOOL

Description
This function tests whether the arrangement A is free from its defining equation Q.

Example 
/**/ use QQ[x,y];	
/**/ A := [x, x-y, y];
/**/ Q_A := product(A);
/**/ IsArrFree(Q_A);    
true

/**/ use QQ[x,y,z];    
/**/ A := [x, x+z, y, 2*y-3*z];
/**/ IsArrFree(product(A));
false
