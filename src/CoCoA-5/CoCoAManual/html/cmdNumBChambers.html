NumBChambers    --    number of bounded chambers of an arrangement of hyperplanes


Syntax 
NumBChambers(A: LIST): INT

Description
This function returns the number of bounded chambers from the list A of hyperplanes in the arrangement.

Example 
/**/ use QQ[x,y];	
/**/ A := [x, x-y, y];
/**/ NumBChambers(A);
0
