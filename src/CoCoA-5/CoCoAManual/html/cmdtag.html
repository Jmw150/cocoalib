tag    --    returns the tag string of an object


Syntax 
tag(E: OBJECT): STRING

Description
If E is a tagged object, this function returns the tag of E; otherwise, it returns the empty string.

Example 
/**/  L := tagged(3,"MyTag");
/**/  type(L);
TAGGED("$TopLevel.MyTag")

/**/  tag(L);
MyTag
