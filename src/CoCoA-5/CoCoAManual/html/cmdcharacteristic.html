characteristic    --    the characteristic of a ring


Syntax 
characteristic(R: RING): INT

Description
This function returns the characteristic of the ring R.

Example 
/**/  use R ::= ZZ/(3)[t];
/**/  S ::= QQ[x,y];
/**/  characteristic(CurrentRing);
3
/**/  S ::= QQ[x,y];
/**/  characteristic(S);
0
