IsPolyRing    --    test whether a ring is a polynomial ring


Syntax 
IsPolyRing(R: RING): BOOL

Description
This function tests whether its argument is a polynomial ring.

Example 
/**/  use P ::= QQ[x,y];
/**/  IsPolyRing(P);
true
/**/  PmodI := NewQuotientRing(P,ideal([x])); // NO, but isom to QQ[y]
false
/**/  IsPolyRing(QQ);
false
