EqSet    --    checks if the set of elements in two lists are equal


Syntax 
EqSet(L: LIST, M: LIST): BOOL

Description
This function returns true if L equals M as sets, otherwise it returns false.

Example 
/**/  L := [1,2,2];
/**/  M := [2,1];
/**/  EqSet(L, M);
true
