type    --    the data type of an expression


Syntax 
type(E: OBJECT): TYPE

Description
This function returns the data type of E.

Example 
/**/ L := [1,"a",2,"b",3,"c"];
/**/ [ X in L | type(X)=INT ];
[1, 2, 3]

/**/  type(type(INT));  -- Type returns a value of type TYPE
TYPE

/**/  CurrentTypes();
[BOOL, ERROR, FUNCTION, ...]
