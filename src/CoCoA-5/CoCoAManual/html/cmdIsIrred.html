IsIrred    --    check if a RINGELEM is irreducible


Syntax 
IsIrred(f: RINGELEM): BOOL

Description
This function tests whether the argument f is irreducible in RingOf(f). The coefficient ring must be a field.

Example 
/**/ FFp := NewZZmod(7);
/**/ use FFp[x];
/**/ f := x^9+x+1;
/**/ IsIrred(f);
true
