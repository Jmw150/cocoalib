IsCommutative    --    test whether a ring is commutative


Syntax 
IsCommutative(R: RING): BOOL

Description
This function tests whether the ring R is commutative.

Example 
/**/  IsCommutative(ZZ);
true
/**/  IsCommutative(NewWeylAlgebra(ZZ,"x,y"));
false
