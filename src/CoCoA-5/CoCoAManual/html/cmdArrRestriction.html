ArrRestriction    --    arrangement of hyperplanes A restricted to a hyperplane


Syntax 
ArrRestriction(A: LIST, H: RINGELEM): LIST

Description
This function returns the list of hyperplanes in the variables [y[1],..,y[n]] obtained by restricting to the hyperplane H of the list A of hyperplanes of an arrangement.

Example 
/**/ use QQ[x,y,z];	
/**/ A := [x, x-z, y-z, z];
/**/ ArrRestriction(A, y-z);
[y[1],  y[1] -y[2],  y[2]]
