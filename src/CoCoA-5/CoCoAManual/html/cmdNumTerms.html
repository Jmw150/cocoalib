NumTerms    --    number of terms in a polynomial


Syntax 
NumTerms(F: RINGELEM): INT

Description
This function returns the number of terms in a polynomial.

Example 
/**/  use R ::= QQ[x,y,z];
/**/  NumTerms((x+y+z)^5) = binomial(3+5-1, 5);
true
