RandomPermutation    --    random permutation (of indices)


Syntax 
RandomPermutation(N: INT): LIST of INT

Description
This function returns a random permutation of the integers 1..N.

Example 
/**/ RandomPermutation(5);
[5,  3,  2,  1,  4]
