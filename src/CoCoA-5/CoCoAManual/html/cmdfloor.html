floor    --    round rational down to integer


Syntax 
floor(X: RAT): INT

Description
This function returns the greatest integer less than or equal to X.

Example 
/**/  floor(0.99);
0

/**/  floor(1.01);
1

/**/  floor(-1);
-1

/**/  floor(-0.01);
-1
