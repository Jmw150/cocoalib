RevLexMat    --    matrix for rev lex term-ordering


Syntax 
RevLexMat(N: INT): MAT

Description
This function returns the matrix defining a standard ordering (which is not a term-ordering!).

Example 
/**/  RevLexMat(3);
matrix(ZZ,
 [[0, 0, -1],
  [0, -1, 0],
  [-1, 0, 0]])
